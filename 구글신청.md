YouTube API 사용 구조:

1. 검색 플로우:

   - 사용자 입력 (학년/주제/의도)
     → OpenAI로 키워드 생성
     → YouTube Search API로 신뢰 채널 검색
     → YouTube Videos API로 메타데이터 수집
     → Gemini AI로 안전성 분석
     → 필터링된 결과 표시

2. API 호출 상세:

   - Search API:
     GET /youtube/v3/search
     파라미터: q={keyword}, channelId={trustedChannel}, maxResults=20

   - Videos API:
     GET /youtube/v3/videos
     파라미터: id={videoIds}, part=snippet,statistics

3. 데이터 처리:

   - 캐싱: Firestore에 7일간 저장
   - 보안: API 키는 환경변수로 관리
   - 에러 처리: 할당량 초과 시 자동 키 rotation

4. 관련 파일:
   - src/utils/youtube.js: YouTube API 호출 로직
   - src/utils/searchCache.js: 캐싱 시스템
   - src/components/VideoRecommendationDirect.jsx: 메인 UI
